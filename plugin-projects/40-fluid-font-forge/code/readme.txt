Advanced fluid typography calculator with CSS clamp() generation for responsive font scaling.

== Description ==

Fluid Font Forge provides an advanced interface for creating responsive typography using CSS clamp() functions. Generate fluid font sizes that scale smoothly between viewport breakpoints.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/fluid-font-forge/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to J Forge > Fluid Font in the admin menu

== Frequently Asked Questions ==

= How do I use the generated CSS? =

Copy the generated CSS from the plugin interface and paste it into your theme's stylesheet.
